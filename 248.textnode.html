Text Node

- Saat kita membuat HTML, kadang kita membuat teks tapa tag
- Saat halaman web tersebut direpresentasikan dalam DOM, text tapa tag bukanlah merupakan Element, melainkan Node
- Lebih tepatnya adalah Text Node https://developer.mozilla.org/en-US/docs/Web/API/Text

Membuat Text Node

- Kita juga bisa membuat Text Node menggunakan method document.create TextNode(string)
- Hasilnya berupa Text yang bisa kita tambahkan ke Node lain
